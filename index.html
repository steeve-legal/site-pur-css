La voie consciente

Sur une route sombre et morne pavée de vestiges de balises spécifiques au navigateur, de DOMs incompatible, de supports cassés et navigateurs abandonnés.

Nous devons effacer l'esprit passé et actuel qui nous conditionne. L'illumination du Web a été obtenue grâce aux efforts inlassables de personnes comme W3C, WaSP, et les principaux créateurs de navigateurs.

Je vous invite à vous détendre et à méditer sur les leçons importantes des maîtres. Commencez à voir avec clarté. Apprenez à utiliser les techniques ancestrales d'une manière nouvelle et revigorante. Devenez un avec le Web.

Donc c'est à quel Sujet?

Il est toujours nécessaire de montrer la puissance du CSS. J'aspire à exciter, inspirer et encourager la participation.

CSS permet un contrôle complet et total sur le style d'un document hypertexte. La seule façon d'illustrer cela d'une manière qui excite les gens est de démontrer ce que cela peut vraiment être, une fois que les rênes sont placées entre les mains de ceux qui sont capables de créer de la beauté à partir de la structure. Les concepteurs et les codeurs ont contribué à la beauté du Web; nous pouvons toujours le pousser plus loin.

Participation

La conception visuelle forte a toujours été mon objectif. Vous voulez modifier cette page, donc de solides compétences sont également nécessaires, mais les fichiers d'exemples sont suffisamment commentés pour que même les novices en CSS puissent les utiliser comme points de départ. Veuillez consulter W3Shools listing des ressources CSS "Guide de ressources CSS pour des didacticiels avancés et des astuces sur l'utilisation de CSS.

Vous pouvez modifier la feuille de style comme vous le souhaitez, ainsi que le fichier HTML. Cela peut sembler intimidant au début si vous n'avez jamais travaillé de cette façon auparavant, mais suivez les liens répertoriés pour en savoir plus et utilisez les exemples de fichiers comme guide.

Téléchargez les exemples des Code HTML et CSS de cette page pour travailler sur une copie localement. Une fois que vous avez terminé votre chef-d'œuvre, téléchargez vos fichier sur un serveur Web sous votre contrôle et partagez.

Benefices

Pourquoi participer? Pour la reconnaissance, l'inspiration et une ressource, nous pouvons tous nous référer à montrer aux gens à quel point CSS peut être incroyable. Ce site est une source d'inspiration à part égale pour ceux qui travaillent sur le Web aujourd'hui, un outil d'apprentissage pour ceux qui le seront demain et une galerie de techniques que nous pouvons tous espérer.

Exigences

Dans la mesure du possible, l'utilisation des niveaux 1 et CSS 1, les niveaux 3 et 4 CSS 3 doivent être limités aux éléments largement pris en charge uniquement, ou des solutions de secours solides doivent être fournies.

Ce CSS concerne des CSS fonctionnels et pratiques et non les dernières astuces de pointe visibles par 2% du public qui navigue. La conception de cette manière montre à quel point les divers navigateurs ont bien implémenté CSS à l'heure actuelle.

En respectant les directives, vous devriez voir des résultats assez cohérents sur la plupart des navigateurs modernes. En raison du grand nombre d'agents utilisateurs sur le Web ces jours-ci, en particulier lorsque vous prenez en compte des mises en page mobiles au pixel près, il peut ne pas être possible sur toutes les plateformes. Ce n'est pas grave, mais testez-en autant que vous le pouvez.

Votre conception doit fonctionner au moins dans IE9 + et les derniers navigateurs Chrome, Firefox, iOS et Android (gérés par plus de 90% de la population). Ceci est un exercice d'apprentissage ainsi qu'une démonstration.

La dernière exigence serait que vous preniez du plaisir à faire cette expérience :)
